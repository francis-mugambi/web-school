Ways of adding CSS to HTML document


Ways of adding CSS to HTML document are:

  • Inline CSS method- This involves setting CSS property using the style attribute inside an HTML element.
  • Internal CSS method- The CSS properties are set inside the <head> section using the <style> element.
  • External CSS method- The CSS properties are written in a separate page from the HTML code and linked to the page using <link> tag

How to add inline CSS.

<h1 style="color:orange;" > An Orange Heading</h1>
<h1 style="color:blue;" >A Blue Heading</h1>
<h1 style="color:red;" >A Red Heading</h1>
<p style="font-size:13px;" >A paragraph with font-size 13px</p>

How to add Internal CSS

<!DOCTYPE html >
<html>
<head>
<style>
body{ background-color: gray; }
h1 {color: blue; }
p {color: red; }
</style>
</head>
<body>
<h1> This is a heading displayed in a blue color</h1>
<p>This is a paragraph displayed in red color.</p>

</body>
</html>

How to add external CSS

<!DOCTYPE html >
<html>
<head>
<link rel="stylesheet" href="styles.css">
</head>
<body>
<h1> This is a heading displayed in a blue color</h1>
<p>This is a paragraph displayed in red color.</p>

</body>
</html>

This is the CSS that is linked to the above page as an external CSS saved as "styles.css"
h1 { color: blue;
}
p { color: red;
}
